# YTVideo-chatbot
A RAG application that extracts youtube video transcripts and answer user questions about the video.
Implemented transcript processing, text chunking, embedding generation and semantic retrieval using LangChain and FAISS.
Integrated an LLM with Streamlit to deliver context aware answers through an interactive chat interface.
